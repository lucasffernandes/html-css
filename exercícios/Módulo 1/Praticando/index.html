Espaçamentos

Para fazer espaçamentos utiliza-se a tag <b>.

Mostrando as tags

Para colocar as setinhas nas tags para serem exibidas ao usuário do site, é necessário utilizar o recurso <TAG ESCOLHIDA>

Adicionadno símbolos

Para adicionar um símbolo, é necessário colocar o e comercial (&) antes da palavra reservada do símbolo, para que seja reproduzido no documento web.

Exemplo:
®
©
™
€
£

Adicionando emogis

Para adionar emogis, é necessário colocar 3 caracteres especificos em html para dizer a linguagem que o valor que vai ser inserido será em hexadecimal.
Os valores são: &#x
Logo após são inseridos os valores hexadecimais respectivos ao emogi desejado.

O site geralmente utilizado para a busca de emogis e seus valores exadecimais é no emogipedia.

Exemplo de emogis buscados:
🖖
😵
🙂‍
❤️

Tag para inserção de imagens

Para inserir imagens em um documento HTML utiliza-se a tag <img>. Esta tag geralmente vem com os atributos:

src
Source, atributo para localizar o documento, seja o diretório de armazenamento local, ou um endereço web da imagem.
alt
É o atributo para descrição de imagens, nele coloca-se uma breve descrição do que se trata a imagen, (essa descrição não aparece ao usuário, somente servirá para buscadores acharem ela na web).

Adicionando favicon no site

Favicons geralmente são suportados somente em formatos .ico, se a iamgem não for desse formato é possível fazer a converção em sites gratúitos e utilizar no site. Para inseri-lo basta somente linkar ele no documento html com a tag <link> no head do documento, depois adicionar o repositório de destino do favicon desejado.
