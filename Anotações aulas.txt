Capítulo 25 / Aula 11 - Criando um menu hambúrger

 * Para se criar um menu de hamburger é preciso adicionar um ícone de hamburger

 * É possível com HTML e CSS criar um icone de hamburger, criar todas as opçoes que vc deseja que seja disponíveis para o usuário, e depois colocar essas opções imbutidas dentro do icone.

   - Mas a iteratividade de clicar no icone e descer as opções é feito com javaScript dentro da função CSS {onclick}.

* Os códigos em javaScript são colocados dentro de um código HTML em ultimmo lugar dentro do corpo do projeto <BODY>

  - O código javaScript pode ser escrito dentro da tag <script>.

  - Após criar a função, basta colocar o nome dessa função (Chamar a função) dentro da definição CSS {onclick}. 
     Ex: onclick = funcaoJS()

* Em javaScript '=' significa "receber"
  - E '==' significa "igual"

* Como o icone é inserido como uma fonte do google fonts, como uma fonte ele é considerado, então como tal o alinhamento é realizado como elemento textual (com 'text-align')

* Com a configuração CSS 'cursor:pointer' o cursor é modificado para ser uma mão quando posicionado em cima de um objeto.

Capítulo 26 Aula 06 – Primeiros passos com Flexbox CSS
--------------------------------------------

* Flexbox é um conceito da nova versão da CSS que visa rescolver problemas relacionados a responsividade
      - flexbox (ou flexible Box) é om módulo da nova versão css 3.O

* Nessa nova versão css tambem tem o recurso de Grid Layout

CSS 2.1

-> Block Layout
-> Inline Layout
-> Table Layout
-> position Layout

CSS 3.0

-> CSS flexible bOX (flexbox) Module Level 01

-> CSS Grid Layout Module Level 1/2

--------

* Não quer dizer que os recursos da CSS level 2.1 não sejam úteis mais, eles ainda podem ser utilizados.

* o flexbox organiza os itens dentro da caixa(Conteiner) filha. ou seja, elá é um recurso adicionado geralmente a caixa pai.

* Para esse recurso funcionar é necessário colocar 'flex: auto' na caixa filha para que funcione o recurso flexbox.

   - organize automaticamente os recursos dentro da caixa de acordo com o tamanho do conteiner.

* somente recebe essas configuracoes os conteiners(Flex-conteiner), os itens(flex-itens) recebem as configurações individuais na css.

Capítulo 26 Aula 06 – Direções e Eixos Flexbox CSS
--------------------------------------------

Eixos (axis)

* Alem da configuração flexbox no container pai, existe outras definições desse recurso(flexbox) disponíveis para utilizar no posicionamento de conteiners. 
   
   - Dentre elas a flex-direction, é uma definição que vem por padrão o valor ":row" - que quer dizer linha -, ou seja, já vem pré configurado a organização das caixas em linha. Nessa configuração os ítens vão ser configurados da esquerda para direita.

      - existe outro valor para a propriedade flex-direction, que seria ":row-reverse", nesse caso, o sentido de organização dos elementos será dá direita para a esquerda.

   - existe tambem o flex-direction: colum; -> que faz a organização de ítens em coluna, de cima para baixo

      - exite nesse caso também o valor colum-reverse; que organiza o sentido dos itens de baixo para cima.

* Para configurar as posições dos elementos, o que precisa ser definido, é a posição dos eixos do conteiner pai, utilizando esse recurso de flexbox, toda caixa pai passa a ter eixos de localização no espaçamento dentro da caixa pai. Podemos definir os eixos como:

  eixo imaginário X = main axis (main-start, main-end)

  Eixo imaginário Y = cross axis (cross-start, cross-end)

* O sentido dos eixos mudam de acordo  com a configuração flex-direction:
   ex:

   flex-direction: row 
     cross-axis (y) cima ->  baixo 
     main-axis (x) esquerda -> direita

   flex-direction: row-reverse
     cross-axis(y) cima -> baixo
     main-axis (x) direita -> esquerda

   flex-direction: colunm
     cross-axis (y) esquerda -> direita
     main-axis (x)  cima -> baixo

   flex-direction: column-reverse
     cross-axis (y) esquerda -> direita
     main-axis (x) baixo -> cima

RESUMINDO

* flex-direction: com o valor 'row' organiza os elementos horizontalmente da esquerda -> direita

- com o valor row-reverse, organiza esses elementos horizontalmente de maneira reversa da direita para a esquerda

* flex-direction: com o valor colunm, organiza os elementos verticalmente de baixo para cima.

- com o valor colunm-reverse, os elementos ficam organizados verticalmente, de maneira reversa, de cima para baixo.
